# SMS114 문자발송 가이드



## 이용안내

```python
누리넷의 모든 서비스는 회원가입 후 정상적으로 이용이 가능하며 만 18세 미만은 가입하실 수 없습니다.
문자메시지, 장문메시지 서비스를 이용하기 위해서는 로그인 후 사이버머니를 충전하셔야 합니다.
충전하신 사이버머니는 소멸기한이 없습니다.
이용하실 수 있는 기능은 아래와 같습니다.

01. 그룹별로 관리할 수 있는 주소록
02. 여러개의 회신번호를 관리할 수 있는 보관함
03. 자주 사용하는 메시지를 보관할 수 있는 메시지보관함
04. 상세한 전송결과내역 및 엑셀파일로 다운로드 제공
05. 문자 전송 시 중복되는 수신자 정보를 걸러내는 필터링 기능
06. 단문메시지(SMS) , 장문메시지(LMS) 모두 한번에 3,000명까지 전송 가능

```

## 문자메시지 보내기

```python
01. 주소록 불러오기
- 개별보기에서 선택 후 주소추가 버튼을 클릭하면 받는사람 리스트로 이동됩니다.
- 그룹보기에서 그룹을 선택 후 주소추가 버튼을 클릭하면 그룹에 속한 주소가 받는사람 리스트로 이동됩니다.

02. 외부데이터(예: 엑셀) 불러오기
- 휴대폰번호만 있는 경우 : 휴대폰번호가 입력된 열(컬럼, Column)만 복사하신 후 붙여넣기 버튼을 
클릭하시면 주소록 상자에 복사한 핸드폰번호가 나타납니다. 자료 확인 후에 받는사람 리스트로 보내기 버튼을 
클릭하시면 받는사람 리스트로 이동됩니다.
- 이름과 휴대폰번호가 있는 경우 : 이름과 휴대폰번호가 각각의 열에 있어야 합니다. 
두 개의 열을 복사하신 후 붙여넣기 버튼을 클릭하시고 나머지는 상기 사항을 참고하세요.

03. 빠른 추가 직접입력
- 이름과 휴대폰번호 혹은 휴대폰번호만 입력하신 후 추가 버튼을 클릭하시면 받는사람 리스트에 추가됩니다.

04. 받는 사람 직접입력
- 한 명에게 보낼 때 사용합니다. 휴대폰번호를 받는사람 란에 입력해 주세요.

05. 최근보낸번호
- 최근에 전송한 번호를 5개까지 보여줍니다. 선택하시면 받는 사람란에 입력됩니다.

```



## 문자메시지 내용 작성 방법

```python
01. 단문: 최대 90 byte(한글 40자)까지 내용을 입력하실 수 있습니다.

02. 단문연속: 단문연속을 클릭하시면 입력란이 3개가 나타납니다.
               각각 내용을 입력하시면 차례대로 5초 간격으로 전송하며 240 byte까지 입력하실 수 있습니다.

03. 장문: 최대 2000 byte(한글 1000자)까지 내용을 입력하실 수 있습니다.

<h3>성명 머지 기능</h3>
문자메시지 내용 입력 화면에 {성명} 을 입력하시면 받는사람 리스트에 등록되어 있는 이름이 {성명}부분을 대신하게 되는 기능입니다.
{성명}머지기능을 클릭하시면 문자메시지 내용 입력 화면에 입력됩니다.
예를 들어 홍길동 010-0000-0000 전송 시 : {성명}님 안녕하세요. → 홍길동님 안녕하세요.
```

## 전송 시기 선택

```python
01. 즉시: 기본으로 선택된 상태입니다. 실시간으로 전송합니다.

02. 예약: 현재 시각을 기준으로 30분 이후로 예약 등록이 가능합니다. 최대 예약기간은 다음 해(익년) 12월 31일까지입니다.
```

## 문자메시지 전송 결과 확인

```python
01. 전송결과 조회 : 전송하신 메시지의 수신결과 및 결과수신시각을 확인하실 수 있으며 파일로 다운로드 하실 수 있습니다.
- 전송성공
- 전송실패: 실패 원인 확인 가능, 삭제 시 해당건수만큼 즉시 재충전됩니다.

02. 전송대기 조회: 전송결과를 수신하지 못했거나 전송결과 수신 후 전송결과 리스트에 등록되지 않은 건을 조회하실 
수 있습니다. 수신대기의 주원인은 아래와 같으며 삭제하실 수 없습니다.

- 수신자 핸드폰의 메세지함이 가득찼을 때
- 수신자 핸드폰 배터리가 꺼져있을 때
- 수신자가 음영지역에 위치해 있을 때
- 수신자 핸드폰이 착신되어 있을 때
- 과도한 트래픽으로 인해 전송이 지연될 때 등

03. 전송예약 조회: 현재시각을 기준으로 30분 후에 예약하신 건에 대해 조회, 수정, 삭제가 가능하며 삭제 시 해당 건수만큼 
즉시 재충전됩니다.
```
